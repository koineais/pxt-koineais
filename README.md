# Fun Utilities - micro:bit拡張機能

micro:bitのLED画面で楽しいアニメーション機能を提供するMakeCode拡張機能です。

## 機能

### 1. Play Fun Animation (楽しいアニメーションを再生)
- micro:bitのLED全体でシンプルなアニメーションを表示します
- 目と口のアニメーション付きです

### 2. Flash LEDs (LEDを点滅)
- 指定した回数だけLEDをハートマーク付きで点滅させます
- パラメータ: 点滅回数 (1～10回)

## 使い方

MakeCode EditorでこのURLから拡張機能を追加してください:
```
https://github.com/koineais/pxt-koineais
```
